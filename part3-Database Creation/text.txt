
Создание своей базы данных

    1. Перейти на phpmyadmin и войти в "базу данных"
    2. Придумать название и написать кодировку cp1251_general_ci - для кирилецы либо utf8-general_ci для других языков
    3. Придумать имя таблицы и количество столбцов

    id - идентификатор пользователя
    Тип: INT - от нуля до безконечности
    Тип: TINYINT - если чисел мало
    Длина: количество значений от 1 до 255
    По умолчанию: какое значение поле принемает по умолчанию
    Сравнение: кодировка 
    Атрибуты: UNSIGNED - числа не могут идти в минус
    Null: пустая ячейка
    Индекс: добовляет индекст 
    Индекс: PRIMARY - может быть один
    Индекс: UNIQUE - поле с именем должнобыть уникальным иначе UNIQUE ставить нельзя
    A_I: при добовлении новой записи увеличивает id на 1
